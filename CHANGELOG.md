# 変更メモ

## 最初にやったこと

- Studio のライブプレビューから表示内容と画像アセットを確認した。
- `index.html` / `styles.css` / `assets/` だけで動く静的ページとして再現した。
- 元ページで使われていた背景画像、犬画像、吹き出し画像、Xアイコンを `assets/` に保存した。

## レイアウトの整理

- 当初は「背景 + 犬4匹 + 吹き出し4つ」の構成が上下に2セットあったため、1セットだけに整理した。
- 4匹の犬それぞれに吹き出しを重ね、ホバーまたはフォーカス時に吹き出しが表示されるようにした。
- 1匹目と右上の犬だけ吹き出し位置がずれていたため、犬と吹き出しの相対位置を統一した。

## PC表示

- 背景画像を1枚絵の基準にして、その上に犬と吹き出しを重ねる構造にした。
- 犬の大きさと位置は、背景に合わせて等倍で変化するようにパーセント指定にした。

主な指定:

```text
犬幅: セクション幅の 16.328%
犬1: left 40%, bottom 0
犬2: left 56.875%, bottom 0
犬3: left 72.8125%, bottom 0
犬4: left 83.672%, bottom 29.444%

吹き出し:
width 84.689%
left -33.971%
top 6.301%
```

## スマホ表示

- 背景と完全同倍率のままだと犬が小さくなりすぎるため、スマホ幅では犬4匹をボタン列として再配置した。
- スマホブレイクポイントは `max-width: 540px`。
- 犬4匹は右下に横並びで配置した。
- 犬は実際の `<button>` 要素に変更し、タップ対象として使いやすい構造にした。

主な指定:

```text
犬4匹を右下横並び
gap 8px
右余白 10px
犬幅 62px〜84px目安
犬の最小タップ領域: min-width 62px / min-height 68px

吹き出し:
width 82%
left -26%
top -2%
```

- 元ページに合わせて薄い白背景、透明度、コピーライト、リンク、Xアイコンを配置した。
